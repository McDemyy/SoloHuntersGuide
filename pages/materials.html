Materials Guide

Crafting Resources · Drop Sources · Item Recipes

Rarity:

Mana Crystal

Common
Source All Gate Ranks
Stack Size 999
Sell Value 120g

⚡ Condensed mana energy. Used to enhance mana-based equipment and brew basic potions.

📍 Obtain: Magic-type monster drops · All rank gates · Guild shop

Ice Fragment

Common
Source C-Rank Gates
Stack Size 999
Sell Value 200g

⚡ Shard of elemental ice. Required to craft Glacier Fang and apply frost enchantments to weapons.

📍 Obtain: Ice Cavern Gate (C-Rank) · Ice Golem drops · Frost Chest

Gold Dust

Rare
Source B–A Rank Gates
Stack Size 500
Sell Value 1,800g

⚡ Refined elemental gold. Used for weapon and armor upgrades (+1 to +10). Essential for most A-rank crafting recipes.

📍 Obtain: Boss chest drops (B–A Rank) · Gate reward box · Auction House

Shadow Stone

Epic
Source A–S Rank Gates
Stack Size 100
Sell Value 12,000g

⚡ Crystallised shadow energy. Required for Shadow Monarch Armor crafting and shadow-type weapon enchantments.

📍 Obtain: Shadow Gate mini-boss drops · A-Rank reward chest · Shadow Monarch Boss

Dragon Scale

Legendary
Source A–S Rank Gates
Stack Size 50
Sell Value 45,000g

⚡ Impenetrable scale from an ancient dragon. Used to forge elite fire-resistant armor and legendary weapons. Extremely rare.

📍 Obtain: Thunder Dragon Boss (A-Rank Gate) · Ancient Dragon Gate (S-Rank)

Void Ore

Mythic
Source S-Rank Gates Only
Stack Size 10
Sell Value 200,000g

⚡ Ore forged from void dimension energy. Required for all Mythic-tier equipment. Only Monarch-class hunters can handle it safely.

📍 Obtain: Void Gate Boss (S-Rank) · Monarch drop · Cannot be traded
